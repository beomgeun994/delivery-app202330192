# 🛵 배달조아

컴퓨터과학개론 기말 프로젝트 — 배달앱 만들기 & 배포

## 🔗 배포 URL
https://delivery-app202330192.vercel.app/

## 🛠 기술 스택
- **Frontend/Backend**: Next.js 15 (App Router)
- **Database**: Neon (PostgreSQL)
- **인증**: JWT (jose) + bcryptjs
- **스타일링**: Tailwind CSS
- **배포**: Vercel

## ✅ 주요 기능
- 회원가입 / 로그인 / 로그아웃
- 식당 목록 + 카테고리 필터
- 메뉴 보기 + 장바구니
- 주문하기 (DB 저장)
- 주문 내역 + 상세 메뉴 확인
- 최소주문 금액 검증
- 주문 상태 표시 (접수/배달중/완료)
